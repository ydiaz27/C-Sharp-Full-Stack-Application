Here are the assignment requirements: 

Form1 should have Buttons: "Add Student" "Add Major" "View Students" and "View Majors".
Each one of these buttons should open a new Form.
Add Student form should open a form that takes the FirstName, LastName, Major. The
ID should be auto generated every time a new Student gets saved starting from ID 1 and
incrementing by 1 every time.
The Add Student Form will have two options: Save and Exit. Save will save the
student information to the Student table in the Database.
Add Major form should open a form that takes Name. The ID should be auto generated
every time a new Major gets saved starting from ID 1 and incrementing by 1 every time.
The Add Major Form will have two options: Save and Exit. Save will save the
major information to the Major table in the Database.
The View Students Form will compose of a DataGridView populated with all the
students in the Student Table from the Database.
The View Majors Form will compose of a DataGridView populated with all the majors in
the Major Table from the Database.
